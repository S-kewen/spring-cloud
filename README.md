# Spring Cloud
![](https://upload.cc/i1/2021/04/06/5dIk0h.jpg)
### SpringCloud是什麽？
* Spring Cloud是一系列框架的有序集合。它利用Spring Boot的開發便利性巧妙地簡化了分布式系統基礎設施的開發，如服務發現註冊、配置中心、消息總線、負載均衡、斷路器、數據監控等，都可以用Spring Boot的開發風格做到一鍵啟動和部署。
